## Обзор

Frosty оптимизирует время автономной работы за счёт заморозки сервисов GMS, применения общесистемных улучшений режима Doze и автоматизации поведения при выключенном экране. Настройка осуществляется через WebUI.

## Функции

- **Заморозка GMS**: Отключение сервисов GMS в 8 категориях.
- **Doze для приложений**: Удалите любое приложение из белого списка энергосбережения Doze от Android. Выбор GMS теперь доступен здесь же, заменяя старый отдельный переключатель GMS Doze.
- **Deep Doze (Глубокий Doze)**: Агрессивные фоновые ограничения для всех приложений (Умеренный / Максимальный).
- **Оптимизация при выключенном экране**: Автоматически отключает выбранные соединения (Wi-Fi, Bluetooth, данные, местоположение) и очищает кэшированные приложения после настраиваемой задержки отключения экрана, а затем восстанавливает всё при разблокировке.
- **Блокировка слежки Google**: Отключает аналитику GMS, телеметрию Clearcut, опросы Phenotype и отслеживание рекламы.
- **Твики ядра**: Оптимизация планировщика, виртуальной машины (VM), сети и отладки.
- **Оптимизатор ОЗУ**: Настраивает лимиты процессов, сжатие памяти и поведение zram.
- **Системные свойства**: Отключение свойств отладки (debug props) для экономии ОЗУ и батареи.
- **Убийство журналов (логов)**: Остановка процессов ведения логов и отладки, истощающих батарею.
- **Тюнинг энергосбережения**: Настройка поведения встроенной функции экономии заряда Android во время её работы.

## Установка

**Требования:** Android 9+, Magisk 20.4+ / KernelSU / APatch, Сервисы Google Play (GMS)

1. Скачайте из раздела [Releases](https://github.com/Drsexo/Frosty/releases).
2. Установите через свой root-менеджер.
3. Перезагрузите устройство.
4. Откройте WebUI, чтобы включить нужные функции.

> [!NOTE]
> Пользователи Magisk могут использовать [WebUI-X](https://github.com/MMRLApp/WebUI-X-Portable/releases) для доступа к WebUI.

## Использование

Откройте WebUI из своего root-менеджера:

- **Системные твики**: твики ядра, системные свойства, отключение размытия, убийство логов, блокировка слежки.
- **Doze**: Doze для приложений с возможностью выбора, Deep Doze с выбором уровня и редактором белого списка.
- **Оптимизация при выключенном экране**: переключатели для каждого соединения, таймеры задержки, восстановление при разблокировке.
- **Категории GMS**: заморозка отдельных групп служб GMS.
- **Тюнинг энергосбережения**: точная настройка поведения режима энергосбережения.
- **Импорт / Экспорт**: резервное копирование и восстановление всей конфигурации.

## Категории GMS

#### Безопасно для отключения
| Категория | Влияние |
|----------|--------|
| 📊 **Телеметрия** | Никакого. Останавливает рекламу, аналитику, слежку. |
| 🔄 **Фон** | Автообновления могут задерживаться. |

#### Может нарушить работу функций
| Категория | Что ломается |
|----------|-------------|
| 📍 **Местоположение** | Карты, навигация, Найти устройство, обмен местоположением |
| 📡 **Связь** | Chromecast, Quick Share, Fast Pair |
| ☁️ **Облако** | Вход через Google, Автозаполнение, пароли, резервное копирование |
| 💳 **Платежи** | Google Pay, бесконтактная оплата NFC |
| ⌚ **Носимые устройства** | Wear OS, Google Fit, фитнес-трекинг |
| 🎮 **Игры** | Достижения Play Игры, списки лидеров, облачные сохранения |

## ЧАВО

**В: Почему мои уведомления задерживаются?**  
О: Doze для приложений и Deep Doze ограничивают фоновую активность. Добавьте свои мессенджеры в белый список Deep Doze через WebUI.

**В: Куда подевался переключатель GMS Doze?**  
О: Теперь он является частью раздела "Doze для приложений" (App Doze). Откройте выбор приложений в App Doze и выберите GMS - эффект тот же, но интерфейс стал единым.

**В: Работает ли модуль без сервисов Google Play?**  
О: Твики ядра, Системные свойства, Отключение размытия, Убийство логов, Оптимизатор ОЗУ и Deep Doze работают без проблем. Для функций GMS, соответственно, требуются сами GMS.

**В: Включено ли что-то сразу после установки?**  
О: Нет. По умолчанию всё выключено. Включайте только то, что вам нужно.
